void LimitedSizeStack.LimitedSizeStack_should.StackDontKeepAllElements ( )

Тест на то, что стек не должен хранить ссылку на элементы, которые уже удалены из него.

Warning
Может вызвать проблемы при запуске на больших объемах данных.
Bug:
Этот тест работает правильно только при использовании FinalizableClass.
Note
Явный вызов сборщика мусора. В обычных программах так делать не нужно почти никогда. Но в этом тесте нам нужно убедиться, что на вытесненные из стека элементы больше не осталось ссылок. Для этого мы вызываем сборщик мусора и проверяем, у скольки объектов сборщик мусора вызвал деструктор.
